Notes on data protection

You* would like to use the Corona-Warn-App ("App") of the Robert Koch Institute ("RKI") to retrieve your test result of an antigen test. In order to retrieve your test result via the App, it is necessary that your test result is transmitted from the testing site to the server system of the RKI.
In short, this is done by the testing site storing your test result, linked to a machine-readable code, on a server of the RKI designated for this purpose. The code is your pseudonym, no further personal information is required to display the test result in the app. However, you can have the display of the test result personalized for you by specifying your name, first name and date of birth.
The code is formed from the intended time of the test and a random number. The formation of the code is done by calculating the aforementioned data with each other in such a way that a back-calculation of the data from the code is no longer possible.
You will receive a copy of the code in the representation of a QR code, which can be read into the app through the camera function of your smartphone. Alternatively, you can also receive the pseudonymous code as an Internet reference ("App Link"), which can be opened and processed by the app. This is the only way to link the test result with your app. With your consent, you can then retrieve your test result using the app. Your test result is automatically deleted from the server after 21 days. If you consent to the transmission of your pseudonymous test result by means of the code to the app infrastructure for the purpose of test retrieval, please confirm this to the testing center staff. You can revoke your consent at any time with effect for the future. Please note, however, that due to the existing pseudonymization, an assignment to your person cannot take place and therefore a deletion of your data will only take place automatically after the 21-day storage period has expired. Details on this can also be found in the "Data protection information" of the Corona-Warn-App of the RKI.

* If you are younger than 16 years old, please discuss the use of the app with your parents or your custodial person.
